# webman

基于webman的api框架，用来撸接口贼爽，常驻内存高性能（支持热重载），RESTful标准响应，thinkORM的Db类，路由灵活，开箱即用

High performance HTTP Service Framework for PHP based on [Workerman](https://github.com/walkor/workerman).
